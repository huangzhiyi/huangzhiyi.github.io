老师推荐的工具软件

2019-10-10
3分钟阅读时长
工具

1.文本编辑：nodepad++

推荐理由：Windows下我用过最快速，文本编码转换最好的文本编辑器。

下载地址：https://notepad-plus-plus.org/downloads/

2. Windows 环境变量编辑器：Rapid Environment Editor

推荐理由：用户界面比较友好的环境变量编辑器，个人使用免费。

下载地址：https://www.rapidee.com/en/download

3. 待办事项管理：微软 TO-DO

推荐理由：免费，快速，支持 Windows、 安卓和iOS设备，而且界面友好，可多平台同步。

下载方法：可以从微软商店搜索下载。移动设备可以从应用商店下载。

4. 云笔记：为知笔记 WizNote

推荐理由：支持 Windows、 安卓和iOS设备，而且界面友好，支持Markdown，目录，附件等功能。最强大的还是它的网页剪切功能，看到想收藏的网页可以一键剪切到云笔记，不用怕以后链接失效内容丢失。虽然是收费的，但是60元每年的服务费用我认为还是比较划算的（一般双11购买还有优惠）。老师从2012年就开始用了。

下载地址：https://www.wiz.cn/

5. 密码管理：KeePass

推荐理由：开源免费的密码管理工具。只需要记住一个密码，用来管理KeePass，其他网站或应用的密码可以使用随机密码。这样可以避免使用同一个密码导致的风险。csdn网站曾因为存储用户的明文密码，后被黑客拖库，导致用户的密码泄露。这些惨痛的教训教育我们必须使用密码管理软件来保护自己，特别是现在网络支付和金融越来越普及的情况下。

配合坚果云来使用可以让密码在云端同步，移动设备也有相应KeePass App可以使用。苹果手机推荐使用奇密。

请一定要从官网下载KeePass：https://keepass.info/

坚果云下载（使用免费，个人版限流）：https://www.jianguoyun.com/

KeePass插件配置：https://zhuanlan.zhihu.com/p/39645975

6. 杀毒和防火墙：火绒安全软件

推荐理由：个人免费，而且静默工作不弹窗，不诱导你安装乱七八糟的软件。如果你受够了XX管家或某数字杀毒软件频繁骚扰你。火绒是一个完美的替代。而且附带的一些弹窗拦截工具可以让你电脑避免尴尬的弹窗。

下载地址：https://www.huorong.cn/person5.html

7. 截图工具：Snipaste

推荐理由：免费，免安装。截图以后可以马上编辑。配置一个全局快捷键，可以让你随时截图并编辑。

下载地址：http://zh.snipaste.com/

8. Windows 命令行工具：Cmder

推荐理由：美观，快速，代码着色，比自带cmd好看不止一两倍。支持ctrl+c，v复制粘贴。多选项卡切换等功能。

下载地址：https://cmder.net

9. 卸载软件工具：Geek Uninstaller

推荐理由：Windows 卸载软件的功能藏的比较深，而且加载奇慢，卸载软件需要一个个卸载。Geek 是个体积小，加载卸载列表非常快的软件。

下载地址：https://geekuninstaller.com

10. 快速查找文件：Listary

推荐理由：个人使用免费，查找文件非常快，而且会自己学习文件名和关键词的关联程度。界面比 Everything 简洁，好看。支持多种文件浏览器如totalCommander，XYPlorer等。我最喜欢的软件之一，已经用了很多年了。因为喜欢所以买了pro版本支持作者。pro版本和免费版其实也没什么区别。

下载地址：https://www.listary.com/download
