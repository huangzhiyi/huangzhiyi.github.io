Git 教程和相关资源

2019-09-10

Git 是目前世界上使用最广泛的分布式版本控制系统。非常多的项目开发都是使用 Git 来管理代码。

  1. 首先我们下载 Git windows 客户端 Git-Windows-64位.exe

  2. 在 gitee.com注册一个账号，创建一个仓库，然后开始学习 git 吧。

  3. 我认为最值得推荐的 Git 的教程，非常浅显易懂，跟着这篇文章学习三天就可以掌握基本的 git 命令。 https://www.liaoxuefeng.com/wiki/896043488029600
