云数据中心基础 Part 1 - 虚拟化软件安装 CentOS

【版本】

当前版本号v20210819

版本修改说明
v20210819初始化版本

【实验名称】

实验1.1 虚拟化软件安装 CentOS

【实验目的】

  • 掌握搭建 CentOS 模板镜像
  • 熟练掌握 Linux命令（vi、tar、mv等等）的使用
  • 掌握 VirtualBox 或 VMWare Workstation的使用
  • 掌握 FinalShell 客户端连接虚拟机

【实验环境】

  • 内存：至少4G
  • 硬盘：至少空余40G
  • 操作系统: 64位 Windows系统。

【实验资源】

  • FinalShell
  • CentOS 7.9系统镜像
  • VirtualBox 6 或 VMWare Workstation
  • Hadoop 安装包
链接：https://pan.baidu.com/s/1pQH-QuWHCs7p8KpO7RiMEA
提取码：heis

【实验步骤】

  1. 安装 VirtualBox 6(也可以选用VMWare Workstation，两者选其一即可)。安装过程略。

  2. 安装 FinalShell。过程略。

  3. 启动 VirtualBox，点击“新建”，新建1台虚拟机。

  4. 类型选择 Linux，版本选择 Red Hat（64-bit）。

  5. 分配 1024M 内存。

  6. 创建虚拟磁盘。

  7. 磁盘映像选择 VDI。

  8. 磁盘选择“动态分配”。

  9. 磁盘分配最大容量 30GB。点击“创建”。

  10. 设置虚拟机。

  11. 加载 CentOS 的安装镜像。

  12. 连接方式选择“仅主机（Host-Only）网络”，网卡选择“VirtualBox Host-Only Ethernet Adapter #2”。

  13. 启动“Tmpl”虚拟机。

  14. 选择“Install CentOS 7”，进行 CentOS 安装。

  15. 虚拟机安装语言选择默认英语。

  16. 时区选择东8区，注意调整时间为你当前安装的实际时间。

  17. 语言支持勾选中文。

  18. 网口设置，启用网口，并设置 Host Name 为 tmpl

  19. 设置网口信息。 

地址：10.0.0.70
掩码：24
网关：10.0.0.254
DNS:223.5.5.5

  1. 回到主界面，进入软件选择界面，选择最小化安装“Minima Install”。

  2. 回到主界面，进入系统安装位置菜单。

  3. 选择手动分区。

  4. 选择标准分区格式

  5. 新建3个分区（Partition）

swap分区：2048M
/boot 分区：300M
/ 根分区：剩余所有空间

  1. 禁用KDUMP。

kdump是Linux内核的一个功能，可在发生内核错误时创建核心转储。当被触发时，kdump会导出一个内存映像，该映像可用于调试和确定崩溃的原因。

  1. 开始安装系统

  2. 设置 root 密码。

  3. 创建 cdc 用户。

  4. 等待系统安装完毕以后，点击重启。

  5. 重启以后，尝试使用 cdc 账户登录。

  6. 进入当前Windows系统的网卡设置，修改虚拟网卡的配置。

  7. 打开 FinalShell，使用 root 用户SSH方式登录虚拟机。

IP:10.0.0.70
端口：22
用户名：root

  1. 为了提升软件安装速度，我们选择使用本地安装镜像作为 yum 的源。首先确保虚拟机加载了 CentOS 的安装光盘镜像。

  2. 调整硬盘为启动的第一顺位。

  3. 解挂目录 /mnt 目录，准备把光盘挂载到此目录。

umount /mnt
  1. 进入源目录，把原有源备份到test目录下。
cd  /etc/yum.repos.d/
mkdir test -p
mv *.repo test
  1. 配置本地源指向光盘镜像的挂载目录。
echo '[local]
name=local
baseurl=file:///mnt
gpgcheck=0'>local.repo
  1. 挂载光盘内容到 /mnt 目录下
mount /dev/cdrom /mnt
  1. 清理源缓存
yum makecache
