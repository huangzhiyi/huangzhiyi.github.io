【实验名称】实验5.1 - 表情温度计项目新增 MQTT 调用接口——修改表情

【实验目的】

  • 掌握设备端 MQTT 调用接口的开发

【实验环境】

  • 内存：至少4G
  • 硬盘：至少空余40G
  • 操作系统: Windows 系统。

【实验资源】

  • iot-thermometer-init.zip
链接：https://pan.baidu.com/s/1z1n-sqrqK_3I2c2GJjfPsw 
提取码：heis

【实验说明】

  1. 使用 MQTTX 客户端向 Mosquitto 代理订阅主题/iot/cloud/thermometer001/receive发送以下格式消息
{"eventId":"0759","eventName":"updateEmojiFace","eventTime":1647155485187,"params":["burn","_(´□`」 ∠)_"]}
  1. 设备端的emoji表中name=‘burn’的表情更新为以下表情
_(´□`」 ∠)_
  1. 并且设备端会向 Mosquitto 代理订阅主题/iot/cloud/thermometer001/receive_reply回复消息是否更新成功。
  • 回复更新成功消息
{"eventId":"0759","eventName":"updateEmojiFace","eventTime":1647155485187,"resMsg":{"errcode":"0","errmsg":"表情更新成功","data":null}}

【实验效果】

  1. 使用 MQTTX 模拟手机端向设备端发送修改表情 MQTT 消息，实现表情的修改，并收到设备端回复消息。

【实验步骤】

  1. 打开iot-thermometer-init项目，完善MqttReceiveServiceImpl.handleSubscribeMsg方法。实现以上实验要求。

提示：可以调用EmojiServiceImpl.updateEmojiFace()方法实现表情更新。
