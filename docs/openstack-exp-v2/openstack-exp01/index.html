任务1 导入并测试模板机

【任务目的】

  • 掌握搭建 RockyLinux 模板镜像
  • 熟练掌握 Linux命令（vi、tar、mv等等）的使用
  • 掌握 VirtualBox 或 VMWare Workstation的使用
  • 掌握使用 SSH 客户端连接虚拟机

【任务环境】

  • 内存：至少4G
  • 硬盘：至少空余40G
  • 操作系统: 64位 Windows系统。

【任务资源】

  • FinalShell
  • RockyLinux 8 系统镜像
  • VirtualBox 7 或 VMWare Workstation
  • Openstack 源
链接：https://pan.baidu.com/s/1fUofELZmtV_tSjjus7-rJQ 
提取码：heis

【任务步骤】

安装 VirtualBox 7

  1. 安装 VirtualBox 7(也可以选用VMWare Workstation，两者选其一即可)。安装过程略。

安装 FinalShell

  1. 安装 FinalShell。过程略。

Windows 宿主机网络设置

  1. 进入当前 Windows 系统的网络适配器设置，修改 VirtualBox 的虚拟网卡VirtualBox Host-Only Ethernet Adapter的配置。

导入虚拟机

  1. 解压openstack_tmpl.zip文件到一个没有中文和空格的路径下。VirtualBox 注册模板机openstack_tmpl，选中openstack_tmpl.vbox文件。
  • 如果使用VMWare Wrokstation，则可以导入openstack_tmpl.ova文件。

  1. 设置网络为Host-Only

  2. 插入光盘文件Rocky-8.8-x86_64-dvd1.iso

  3. 启动虚拟机，使用以下用户名和密码登录

用户名：root
密码：123456

  1. 配置 FinalShell 连接虚拟机。

  2. 测试源是否正常工作。

dnf repolist
dnf makecache
  • 正常会返回类似以下内容。
仓库 id                仓库名称
local-appstream        Rocky Linux 8 - Local - AppStream
local-baseos           Rocky Linux 8 - Local - BaseOS
local-openstack-yoga   Openstack Yoga
  1. 测试 Python 是否正常工作。
python -V
  • 正常会返回 Python 的版本
Python 3.6.8
  1. 关闭 Virtualbox 虚拟网卡的 DHCP 服务（这个服务会干扰 OpenStack 分配给虚拟机的 IP 地址）。
