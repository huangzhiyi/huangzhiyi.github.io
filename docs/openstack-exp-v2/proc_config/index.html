怎么快速对比两个配置文件

  1. 下载 WinMerge （WinMerge-2.16.46-x64-Setup.exe），并进行安装。

  2. 打开 WinMerge，顶部菜单选择“文件”-> “打开”。

  3. 选择2个需要对比的文件打开对比，或者可以把2个文件拖到 WinMerge 的窗口。

  4. 查看标黄的行的对比内容。
