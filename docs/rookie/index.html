计算机菜鸟丰羽计划

2024-09-12
1分钟阅读时长
rookie

做这个系列的初衷，是为了帮助更多刚进入计算机领域学习的初学者。希望能够提供更好的学习方法，更有效率的软件，更优的学习路径和普及计算机的一些基础知识。

在我担任高校教师的期间，我发现很多计算机专业低年级的学生甚至无法用双手熟练打字，还是使用2只食指的“螳螂拳”式打字。还有非常多的基础问题，诸如无法正确安装和卸载应用软件、无法正常的描述一个专业的问题、英文专业术语不会读等等。这些基础的问题会很大程度影响我的教学，所以我启动了一个名为“计算机菜鸟丰羽计划”，意为帮助学习计算机的菜鸟不断提升，逐渐丰满自己的“羽毛”，希望帮助更多的初学者快速提升，迈入计算机专业的门槛。
