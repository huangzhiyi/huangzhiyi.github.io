选择输入法和软件

输入法种类主要有2种，拼音输入法和五笔输入法，以现在的拼音教育普及率来说，拼音输入法应该是初学者的首选。

常用拼音输入法软件：

  • 搜狗拼音输入法
  • 百度输入法
  • 讯飞输入法

键盘打字的正确手型

当我们刚开始学习打字的时候就应该使用正确的手型，正确的手型不但可以提升我们的打字效率，还可以降低我们手指的疲劳度。左手和右手的食指分别放在F键和J键，通常这2个键都有一个小的凸起，帮助我们盲打的时候确保手型放在正确的键位上。

在敲击键盘的时候，请充分利用所有的手指，而不是只依赖食指输入。这种“二指禅”的输入方式很难提升打字的效率。

如何打出大写字母和标点符号

当我们需要打出单个大写字母或者键盘上的标点符号的时候，按住Shift键+相应按键即可。

例如：

Shift键 + b键 输出 B

Shift键 + 2键 输出 @

当我们需要连续输出大写字母的时候，我们可以按下Caps键(Caps Lock键)，此时键位指示灯会亮起，再敲击键盘的字母键即可输出大写字母。

如何练习提升打字速度

  • （1）确保手型的正确，每个手指负责敲哪些键也要明确，哪怕一开始输入比较慢。
  • （2）记住键位以后，一定要坚持盲打，即不要一直看着键盘来打字，只有偶尔忘记的情况下可以看一下。
  • （3）通过打字训练软件或者网站来坚持练习，通过闯关的方式提升自己的成就感。这里推荐TypingClub这种闯关式的打字网站来练习。
