【实验名称】

实训 4.1 设计“表情温度计”功能（使用JPA）

【实验目的】

  • 掌握数据库的设计和脚本编写
  • 掌握 JPA 的使用

【实验环境】

  • Windows 7 以上64位操作系统

【实验资源】

  • JDK 8 - Java Development Kit是 Oracle 公司针对Java开发人员发布的免费软件开发工具包，是 Java 开发必备的开发工具。
  • Intellij IDEA - 业界简称IDEA，是 jetbrains 公司推出的和 Eclipse 齐名的 Java 集成开发环境（IDE）。
  • Apache Maven - Apache Maven，是一个软件项目管理及自动构建工具，由Apache软件基金会所提供。是 Java 构建打包最广泛使用的工具。
  • MySQL 世界上最受欢迎的开源关系数据库之一。
  • HeidiSQL，开源数据库客户端。
  • Cmder 是一个美化的 Windows 命令行终端，支持 ctrl-V 粘贴。
  • Notepad++ 是一个开源文本编辑器，支持代码着色等多种功能，比 Windows 原生文本编辑器功能要丰富很多。
  • Flyway 是一个管理数据库脚本迁移的工具。
  • Eclipse Mosquitto 是一个开源的 MQTT 服务器。支持5.0、3.1.1和3.1的MQTT协议。
  • MQTT.fx 是一个基于 JavaFX 的 MQTT 客户端.

  1. 使用 IDEA 打开 iotweb 项目。

  2. 接下来我们将设计表情温度计的功能。简单来说，就是根据某个地区的某个时刻的检测到的温度，来显示一个相应的 Emoji 表情。

  1. 创建com.example.iotweb.entity.TempConfig类，作为tempconfig表的实体类。
import javax.persistence.*;

@Entity
@Table(name="tempconfig")
public class TempConfig {

    @Id
    @GeneratedValue(strategy= GenerationType.IDENTITY)
	@Column(name="id")
    private Long id;

    @Column(name="dist")
    private String dist;

    @Column(name="mint")
    private Integer mint;

    @Column(name="maxt")
    private Integer maxt;

    @Column(name="emojiname")
    private String emojiName;

    public Long getId() {
        return id;
    }

    public void setId(Long id) {
        this.id = id;
    }

    public String getDist() {
        return dist;
    }

    public void setDist(String dist) {
        this.dist = dist;
    }

    public Integer getMint() {
        return mint;
    }

    public void setMint(Integer mint) {
        this.mint = mint;
    }

    public Integer getMaxt() {
        return maxt;
    }

    public void setMaxt(Integer maxt) {
        this.maxt = maxt;
    }

    public String getEmojiName() {
        return emojiName;
    }

    public void setEmojiName(String emojiName) {
        this.emojiName = emojiName;
    }
}

  • getEmojiTempByDistAndTemp 根据参数地区（dist）和温度（temp）返回表情（Emoji face）、表情名称（Emoji name）、dist和temp，例如： http://localhost:8099/getEmojiTempByDistAndTemp?dist=zhanjiang&temp=27 能够返回以下字符串

  • 新建src/main/resources/com/example/iotweb/mapper/EmojiTempMapper.xml，用于配置 Mapper 方法对应的 SQL 和返回结果类型等。注意此配置文件是XML 格式，所以 SQL中出现的大小于号需要进行转义。

大于号写成 > 小于号写成 <

<?xml version="1.0" encoding="UTF-8" ?>
<!DOCTYPE mapper PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN" "http://mybatis.org/dtd/mybatis-3-mapper.dtd">
<mapper namespace="com.example.iotweb.mapper.EmojiTempMapper">
    <select id="getEmojiTempByDistAndTemp" resultType="com.example.iotweb.vo.EmojiTempVo">
        SELECT e.name as emojiName,e.face as emojiFace,t.dist,#{temp} as temp FROM tempconfig t LEFT JOIN emoji e ON t.emojiname=e.name WHERE t.dist=#{dist} AND t.maxt &gt; #{temp} AND t.mint &lt;= #{temp} LIMIT 1
    </select>
